# St. John de Britto's Church — Frontend

A modern React + Vite frontend for the Church Management Platform.

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

The frontend will start at `http://localhost:5173`.

## Tech Stack

- React 19 with Vite
- Tailwind CSS v3
- React Router DOM v7
- Framer Motion for animations
- Axios for API calls
- React Hook Form + Zod validation
- i18next for internationalization (English + Tamil)

## Build for Production

```bash
npm run build
npm run preview
```
